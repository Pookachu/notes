#Logic-Gate
### Overview
Negates any signal fed into it, flipping it.
#### Truth table

| Input X | Output X' |
| ------- | --------- |
| 1       | 0         |
| 0       | 1         |
#### Boolean Algebra Expression
Symbols:
	1. Bar: $\bar x$
	2. Apostrophe: $x'$ 
Example:
